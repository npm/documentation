# utility-shubham-palav

A lightweight JavaScript utility library with commonly used helper functions.

## Installation
```bash
npm install utility-shubham-palav

Usage
sum

Adds multiple numbers and returns the total.

import { sum } from "utility-shubham-palav";

sum(1, 2, 3); // 6

capitalizeWords

Capitalizes the first letter of each word in a sentence.

import { capitalizeWords } from "utility-shubham-palav";

capitalizeWords("hello world"); 
// "Hello World"
