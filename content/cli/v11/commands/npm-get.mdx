### Synopsis

```bash
npm get [<key> ...] (See `npm config`)
```

Note: This command is unaware of workspaces.

### Description

Get a value from the npm configuration

### Configuration

#### `long`

- Default: false
- Type: Boolean

Show extended information in `ls`, `search`, and `help-search`.
